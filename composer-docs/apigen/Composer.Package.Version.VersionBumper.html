    Class Composer\Package\Version\VersionBumper

    Methods
    public bumpRequirement(ConstraintInterface $constraint, PackageInterface $package): string
    Given a constraint, this returns a new constraint with the lower bound bumped to match the given package's version.

    Given a constraint, this returns a new constraint with the lower bound bumped to match the given package's version.

    For example:

    • ^1.0 + 1.2.1 -> ^1.2.1
    • ^1.2 + 1.2.0 -> ^1.2
    • ^1.2.0 + 1.3.0 -> ^1.3.0
    • ^1.2 || ^2.3 + 1.3.0 -> ^1.3 || ^2.3
    • ^1.2 || ^2.3 + 2.4.0 -> ^1.2 || ^2.4
    • ^3@dev + 3.2.99999-dev -> ^3.2@dev
    • ~2 + 2.0-beta.1 -> ~2
    • dev-master + dev-master -> dev-master
        • 1.2.3 -> >=1.2.3
