array loadPackages(array $packageNameMap, array $acceptableStabilities, array $stabilityFlags, array $alreadyLoaded = [])

Returns list of registered packages with the supplied name

  • The packages returned are the packages found which match the constraints, acceptable stability and stability flags provided
  • The namesFound returned are names which should be considered as canonically found in this repository, that should not be looked up in any further lower priority repositories

array[] search(string $query, int $mode = 0, string|null $type = null)

Searches the repository for packages containing the query

Parameters

string $query

search query, for SEARCH_NAME and SEARCH_VENDOR regular expressions metacharacters are supported by implementations, and user input should be escaped through preg_quote by callers
int $mode

a set of SEARCH_* constants to search on, implementations should do a best effort only, default is SEARCH_FULLTEXT
string|null $type

The type of package to search for. Defaults to all types of packages

Return Value

array[]

an array of array('name' => '...', 'description' => '...'|null, 'abandoned' => 'string'|true|unset) For SEARCH_VENDOR the name will be in "vendor" form
